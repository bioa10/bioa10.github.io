Character Sheet



This is a simple character sheet that I have created using Google Sheets to have an editable character sheet saved online. I have included functionality for the sheet to do as much of the math for the character as I could. Currently my character sheet only supports Aspect character creation.

In order to use this, click the button below to go to the Google Sheets page for the character sheet. In the top left press File>Make a copy and choose a name for the file. Once you have a copy, you can fill it out as normal. Blue fields indicate cells that are being used for math calculations and should not be edited. Dark gray fields indicate fields that your character does not have access to yet.
